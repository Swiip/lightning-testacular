Testacular

Spectacular Test Runner for JavaScript.

Testacular n'est pas

  • Un jeu de mot pourri
  • Yet Another JS Test Framework
  • Un composant d'AngularJS

Testacular est

  • Un nom qui passe mal en français
  • Yet Another JS Test Runner
  • Un outil développé par l'équipe d'AngularJS

Schéma de principe

Configuration

basePath = '../';
files = [
    JASMINE,
    JASMINE_ADAPTER,
    'app/lib/angular/angular.js',
    'app/lib/angular/angular-*.js',
    'test/lib/angular/angular-mocks.js',
    'app/js/**/*.js',
    'test/unit/**/*.js'
];
autoWatch = true;
browsers = ['Chrome', 'Firefox'];
junitReporter = {
    outputFile: 'test_out/unit.xml',
    suite: 'unit'
};

Lancement


#!/bin/bash

BASE_DIR=`dirname $0`

export CHROME_BIN=/usr/bin/chromium-browser
export FIREFOX_BIN=/usr/bin/firefox

testacular start $BASE_DIR/../config/testacular.conf.js $*
